## 1. 서비스 개요

- 서비스명: LocalHub
- 대상 권역: 대전/충청권
- 인증 방식: 없음 (익명 커뮤니티, 게시글·댓글 단위 비밀번호로만 수정·삭제 권한 확인)
- 비밀번호는 평문 저장·비교 (교육 목적의 의도된 설계, 암호화하지 않음)

## 2. 필수 기능

| 구분 | 기능 | 설명 |
|------|------|------|
| 커뮤니티 | 게시글 목록 조회 | 카테고리·검색·페이지네이션 |
| 커뮤니티 | 게시글 상세 조회 | 조회수 증가 포함 |
| 커뮤니티 | 게시글 작성 | 제목·내용·수정용 비밀번호 |
| 커뮤니티 | 게시글 수정/삭제 | 등록 비밀번호 일치 확인 |
| 챗봇 | 자연어 질의응답 | `POST /api/chat`, OpenAI API 연동 |
| 챗봇 | 채팅 UI | 플로팅 버튼, 대화 히스토리, 모바일 대응 |

## 3. 선택 기능

- (팀 결정 필요, 최소 1개) 예: 지도 시각화 / 데이터 시각화 대시보드 / 축제 캘린더 등
- 선택 기능 구현 시 활용 데이터의 라이선스를 `data/SOURCE.md`에 추가 기재

## 4. 데이터 출처 및 라이선스

`data/SOURCE.md`, `data/SCHEMA.md` 참조.

## 5. 화면 목록

- 홈 (배너 + 최근 게시글 + 챗봇 플로팅)
- 게시판 목록
- 게시글 상세 (비밀번호 확인 모달)
- 게시글 작성/수정
- 챗봇 위젯 (접힘/펼침)

## 6. API 명세 (초안)

| Method | Path | 설명 |
|--------|------|------|
| GET | `/api/posts` | 게시글 목록 (category, keyword, page, size) |
| GET | `/api/posts/{id}` | 게시글 상세 |
| POST | `/api/posts` | 게시글 작성 |
| PUT | `/api/posts/{id}` | 게시글 수정 (비밀번호 확인) |
| DELETE | `/api/posts/{id}` | 게시글 삭제 (비밀번호 확인) |
| POST | `/api/posts/{id}/verify-password` | 비밀번호 검증 |
| POST | `/api/chat` | 챗봇 질의응답 |
